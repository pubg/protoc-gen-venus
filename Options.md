# Plugin Options
| Option             | Description     | Type    | Default   | Required |
|--------------------|-----------------|---------|-----------|----------|
| expose_all         | 모든 메시지를 노출할지 여부 | Boolean | false | false    |
| output_file_suffix | 출력 파일의 접미사      | String  | .venus.json | false    |
| pretty_output      | 출력 파일을 포맷팅할지 여부 | Boolean | true      | false    |
